## Фокус
- **Интерактивные веб-сцены**: быстрые прототипы для браузера, ориентированные на “открыть и сразу увидеть”.
- **WebXR / WebAR**: эксперименты с XR-взаимодействием и сценами под мобильные/браузерные пайплайны.
- **Realtime сборки**: публикация билдов (в т.ч. WebGL) в репозитории как артефакта/демо.
- **Тестовые стенды**: маленькие репы под конкретный API/интеграцию, чтобы быстро проверить гипотезу.

---

## Технологии

**Основное**
- **HTML / CSS / JavaScript** — минимальный стек для демо, прототипов и публикации.
- **WebGL / WebXR** — интерактив и XR-сцены в браузере.

**Движки / сборки**
- **Unity WebGL** — репозитории с типичной структурой сборки (Build/TemplateData).
- **Unreal Engine (UE/UE4)** — проекты/артефакты, включая `.uproject` и HTML5-вывод.

---

## Сейчас
- Упаковываю прототипы в **малые репозитории**, чтобы их проще было переиспользовать и показывать.
- Собираю **демо-пейджи** (в т.ч. ветки под публикацию) для быстрых просмотрoв без установки окружения.
- Делаю **набор тестов** под конкретные веб-API/интеграции (узкие репы вместо монолита).
- Слежу за тем, чтобы сборки были **переносимыми**: открыть → проверить → повторить.
