## Collaboration

**RU**
- Лучший старт — **Issue**: цель, устройство/браузер, шаги, ожидаемое/фактическое поведение, скрин/видео (если есть).
- Для улучшений — **Pull Request** с коротким описанием: что и почему меняется.
- Для обсуждения идеи/подхода — **Discussions** (если включены в репозитории) или Issue с меткой “discussion”.

**EN**
- Best entry point — open an **Issue**: goal, device/browser, steps, expected vs actual, screenshot/video (if any).
- Improvements — send a **Pull Request** with a short “what/why”.
- Ideas/approach talk — **Discussions** (if enabled) or an Issue tagged as “discussion”.
